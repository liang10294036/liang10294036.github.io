EasyPermissions轻松搞定动态权限申请

发表于 2018-03-23 | 分类于 Android

android 6.0的发布对权限系统进行了改变，部分“Dangerous Permission”需要在运行时询问申请，这中间涉及到与用户交互问题，例如 用户操作 同意 拒绝 不再询问 都要一一做出响应，很是繁琐， 目前google已经推出了开源项目Easypermissions来简化这一操作
